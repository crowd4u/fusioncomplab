雙峰祭企画「ChatGPT時代の就職、仕事、働き方」は終了しました。
対面参加とライブ配信のハイブリット形式で開催させていただきましたが、多くの方にご参加いただき、本当にありがとうございました。

ご参加いただいた皆さんは、講演内容やパネルディスカッションを通じて、ChatGPT時代に私たち人間がどう生きていくべきかが少しわかってきたのではないでしょうか？
個人的には、ChatGPTが人間のできることを代わりにやってくれるようになっていく中で、人間が「次に何ができるようになるのか」が大切ではないのかと思いました。

講演してくださった専門家の皆さま、企画準備をお手伝いいただいた皆様、そして、ご参加いただいた皆様、改めてありがとうございました。

融合知能デザイン研究室4年 田村匠


(画像クリックで拡大します)

オンラインワーク、クラウドソーシング、ChatGPT、ロボットなど僕たちの働き方を取り巻く環境はこの10年で大きく変化している。10年後、20年後の未来を見据えた就職、仕事、働き方、起業等についてどうあるべきか、この問題に関連する第一線の方々を招へいし、筑波大学生、筑波技術大学生の意見を集約するchatTKBと一緒に議論してもらいます。未来の働き方を一緒に考え、自分の進路や働き方のヒントにしよう！

イベント名称

ChatGPT時代の就職、仕事、働き方

イベント概要

オンラインワーク、クラウドソーシング、ChatGPT、ロボットなど僕たちの働き方を取り巻く環境はこの10年で大きく変化している。10年後、20年後の未来を見据えた就職、仕事、働き方、起業等についてどうあるべきか、この問題に関連する第一線の方々を招へいし、筑波大学生、筑波技術大学生の意見を集約するChatTKB*と一緒に議論してもらいます。未来の働き方を一緒に考え、自分の進路や働き方のヒントにしよう！

*ChatTKB：筑波大学の学生が本イベントに向けて独自開発した、パネルディスカッション中に寄せられた参加者の意見の集約を行うシステム
